    (async) search_market(page, limit, filter)

    Search offers from Flea Market.

    Parameters:
    Name Type Description
    page Number

    starting page, example: start searching from page 0.
    limit Number

    limit how many results to show. Example: 15.
    filter Object

    Market Filter

    Properties
    Name Type Attributes Default Description
    sort_type Number <optional>
    		 5

    ID = 0, Barter = 2, Mechant Rating = 3, Price = 5, Expiry = 6
    sort_direction Number <optional>
    		 0

    Ascending = 0, Descending = 1
    currency Number <optional>
    		 0

    All = 0, RUB = 1, USD = 2, EUR = 3
    min_price Number <optional>
    		 0

    Won't show offers below this number
    max_price Number <optional>
    		 0

    Won't show offers higher than this number
    min_condition Number <optional>
    		 0

    Won't show offers where item won't match minium condition
    max_condition Number <optional>
    		 100

    Won't show offers where item won't match maxium condition
    expiring_within_hour Boolean <optional>
    		 false

    Show items that are expiring within hour
    hide_bartering_offers Boolean <optional>
    		 true

    Should we hide bartering offers
    owner_type Number <optional>
    		 0

    Any owner = 0, Listed by traders = 1, Listed by players = 2
    hide_inoperable_weapons Boolean <optional>
    		 true

    Hide weapons that are inoperable
    handbook_id String <optional>
    		 ""

    item id you are searching
    linked_search_id String <optional>
    		 ""

    if you are performing linked item search, include item id
    required_search_id String <optional>
    		 ""

    if you are performing required item search, include item id
